# Архитектура кластера Kubernetes

## Компоненты

### 1. Load Balancer Service
- Тип: LoadBalancer  
- Порт: 80
- Внешний IP: 127.0.0.1
- Назначение: Балансировка входящего трафика

### 2. Horizontal Pod Autoscaler
- Минимум подов: 3
- Максимум подов: 15
- Целевая CPU утилизация: 70%

### 3. Deployment
- Образ: nginx:1.27-alpine
- Реплики: 3 (исходно)
- Ресурсы: 50m CPU, 64Mi RAM
